clone#

TubeSelectionRule.clone(*, unlink=False)#

Create a new unstored object with the same properties.

Parameters:

unlink (bool, default: False) – If True, remove all links to other objects. This can be used to store the object to another model, where the links would be invalid.

Return type:

TypeVar(_T, bound= TreeObjectBase)
